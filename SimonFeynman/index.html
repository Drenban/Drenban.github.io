About

CineShader is a real-time 3D shader visualiser leveraging the Shadertoy.com API to bring shader artworks into a cinematic 3D environment.

Started as a demo to explain procedural noise to Lusion clients, it evolved into a public tool for Shadertoy users to showcase their shaders differently.
